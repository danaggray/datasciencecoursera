## Data Transformations ##
- Read data activities and features into data frames
- 
- Merge training and test sets Test and training data, subject ids and activity ids are merged into a single data set. 
- 
- Subset the data by mean and standard deviation
- 
- Set descriptive activity names
- 
- Set label variables appropriately with Tidy suitable names
- 
- Create a tidy data set from the intermediate data set 
- 
- Output data set to tidydata.txt file
